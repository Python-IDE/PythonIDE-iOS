appui

声明式原生界面、状态、布局、导航、控件、API 参考与示例。

开始

先建立 appui 的基本写法和质量标准。

appui 概览用 Python 编写原生 MiniApp 页面、状态、导航和交互。快速上手用四个可预览示例掌握 body、State、控件和导航。运行时选择判断该用 appui、widget、ui、scene 还是 console。AppUI UI 模式按页面类型选择 Form、List、NavigationStack、TabView 和媒体承载。AppUI API 地图按任务选择入口、状态、布局、控件、导航、媒体和图表 API。MiniApp 开发排错排查入口、状态、按钮、输入、列表、布局和系统能力问题。

指南

按真实界面结构学习状态、布局、导航、交互和性能。

appui 思维方式从命令式 UI 切到 State 驱动的声明式 View 树。状态管理State、ReactiveState、Timer 和批量更新的使用边界。布局系统用原生容器搭列表、表单、网格和可滚动页面。列表与表单List、ForEach、Form、Section、搜索、刷新和滑动操作。导航与页面结构TabView、NavigationStack、NavigationLink 和 toolbar 的组合方式。交互与回调按钮、输入、on_change、sheet 和行级回调的可靠写法。媒体与系统集成图片、相册、相机、地图、视频和 WebView 的页面模式。图表与绘图Chart、Canvas、DrawingContext 和 Path 的使用模式。深色模式与语义色语义颜色、材质背景、强制色彩方案和符号配色。动画animate、animation、transition、matched geometry 和 phase animator。热重载hot_reload 的启用方式、状态保留和调试边界。实时快路径高频状态更新、二进制属性通道和实时 UI 的边界。性能与实时界面普通 State、Timer、ReactiveState 和实时快路径的选择。

API 参考

按任务和组件查找公开 AppUI API。

入口函数run、dismiss、animate、bind、grid item 等模块级函数。状态 APIState、ReactiveState、NavigationPath、Timer 和 Ref。布局 APIStack、ScrollView、List/Form、Grid 和 GeometryReader。堆叠布局 APIVStack、HStack、ZStack、Lazy stacks、Spacer 和 Divider。滚动 APIScrollView、ScrollViewReader、滚动方向和定位锚点。网格 APILazyVGrid、LazyHGrid、Grid、GridRow 和轨道辅助函数。几何与特殊布局 APIGeometryReader、ViewThatFits、Group、Overlay 和 SafeAreaInset。控件 APIButton、TextField、Toggle、Picker、Slider 等控件签名。导航 APINavigationStack、NavigationLink、TabView、ToolbarItem。数据展示 APIList、ForEach、Form、Section、Table、ProgressView 等数据展示视图。呈现 APIalert、sheet、popover、confirmation dialog 和刷新动作。修饰符 API外观、布局、导航、交互和可访问性修饰符。环境值 APIenvironment_value、color_scheme、locale、layout_direction 和动态字体。媒体 APIImage、AsyncImage、PhotoPicker、CameraPicker、VideoPlayer、WebView 和 MapView。图表与画布 APIChart、Canvas、DrawingContext 和 Path。形状 APIRectangle、RoundedRectangle、Circle、Capsule、Ellipse、Color 和 ToolbarItem。
