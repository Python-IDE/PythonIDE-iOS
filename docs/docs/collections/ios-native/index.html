iOS 原生模块

权限、存储、设备、媒体、连接、通信与端侧智能。

快速上手

原生模块总入口：入口模型、Python 模块表、AppUI 桥接、专题页与权限矩阵。

iOS 原生能力相册、联系人、权限、相机、控制台和安全存储。

基础与权限

权限、持久化、钥匙串、剪贴板与弹窗反馈。

permission统一查询权限状态，并请求位置或通知权限。storageUserDefaults 持久键值存储。databaseMiniApp 作用域内 SQLite 数据库和 JSON collection。keychainiOS 钥匙串安全存储。clipboard读取、写入和清空系统剪贴板。console原生弹窗、HUD 和控制台样式。dialogs原生对话框、表单和日期选择。

设备与传感器

设备信息、定位、运动、触觉、生物认证与健康数据。

device设备、屏幕、电池和系统状态。location定位、指南针和地理编码。motion设备运动、陀螺仪、磁力计和气压计。haptics触觉反馈和 Core Haptics 模式。biometricFace ID / Touch ID 验证。healthHealthKit 授权、查询和体重写入。

界面与系统

文件选择、分享、通知、通讯录、日历与 Live Activity。

file_picker系统文件选择器与 AppUI FileImporter。share系统分享面板与 AppUI ShareLink。font_picker系统字体选择器。notification本地通知权限、定时通知和角标管理。live_activity锁屏和 Dynamic Island 的 Live Activity 控制。contacts联系人读取、编辑、选择器和 vCard 导入导出。calendar_events日历事件、提醒事项和 EventKit 权限。

媒体

相册、录制、播放、语音与音视频合成。

采集

photos相册选择、拍照、保存和资源查询。camera系统相机拍照与 AppUI CameraPicker。video_recorder摄像头录像保存为文件。audio_recorder麦克风录音、暂停继续和电平监测。

播放

sound音效播放、长音频播放器和静音开关行为。avplayer音视频加载、播放控制和原生播放器。music_player队列式原生音乐播放器、锁屏控制和播放恢复。musicApple Music 播放控制与曲库搜索。now_playing锁屏与控制中心的正在播放信息。audio_session配置共享 AVAudioSession。

语音

speech文字转语音和语音列表。speech_recognition语音转文字：实时听写和音频文件转写。shazam听歌识曲：麦克风或音频文件匹配 Shazam 曲库。

合成

media_composer视频合并、配音与导出。

视觉与文档

OCR、Vision 检测、Core ML、PDF 与二维码。

vision使用 Vision 从图片字节中识别文字。vision_helper人脸、条码、矩形与图像分类。coremlCore ML 模型加载、图片推理和模型信息读取。qrcode生成二维码图片（读取用 vision_helper）。pdf创建、读取、渲染和预览 PDF 文档。

连接与后台

HTTP、WebSocket、天气、蓝牙、后台任务与本地服务。

联网

network原生 HTTP 请求、连接状态和下载。websocket原生 WebSocket 连接、收发和回调。weather当前天气与每日/每小时预报（WeatherKit）。

近场

bluetoothBLE 扫描、连接、服务发现和特征读写。ble_peripheralBLE 外设广播模式。

后台

background后台任务、保活与 BGTask 调度。background_download后台 URLSession 大文件下载。

本地服务

http_server本地 HTTP 文件服务器。sshSSH 远程命令与 SFTP 文件传输。

通信与智能

邮件、短信、翻译、NFC、闹钟、内购与端侧 AI。

mail系统邮件编写器发送邮件。message系统信息编写器发送短信或 iMessage。translation设备端文本翻译。nfcNFC 标签读取和 NDEF 写入。alarm系统闹钟调度（AlarmKit，iOS 18+）。storekit应用内购买与订阅（StoreKit 2）。foundation_models设备端大模型文本生成（Foundation Models，iOS 26+）。assistant设备端助手与内置设备工具（iOS 26+）。
