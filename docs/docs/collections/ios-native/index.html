iOS 原生模块

权限、存储、设备、传感器、媒体、网络和系统服务模块。

快速上手

先了解原生模块的范围、权限和总览。

iOS 原生能力相册、联系人、权限、相机、控制台和安全存储。全部模块总览按分类查看全部 30+ 内置 Python 模块。原生能力入口MiniApp 常用原生能力入口，不是完整模块总表。

设备与传感器

设备信息、定位、运动、触觉、生物认证和健康数据。

device设备、屏幕、电池和系统状态。location定位、指南针和地理编码。motion设备运动、陀螺仪、磁力计和气压计。haptics触觉反馈和 Core Haptics 模式。biometricFace ID / Touch ID 验证。healthHealthKit 授权、查询和体重写入。

媒体与视觉

照片、音视频、语音、OCR 和 Vision 运行时。

photos相册选择、拍照、保存和资源查询。avplayer音视频加载、播放控制和原生播放器。sound音效播放、长音频播放器和静音开关行为。speech文字转语音和语音列表。vision_helper人脸、条码、矩形与图像分类。vision使用 Vision 从图片字节中识别文字。coremlCore ML 模型加载、图片推理和模型信息读取。

网络与数据

HTTP、WebSocket、BLE 和后台任务。

network原生 HTTP 请求、连接状态和下载。websocket原生 WebSocket 连接、收发和回调。bluetoothBLE 扫描、连接、服务发现和特征读写。background后台任务、保活与 BGTask 调度。

系统服务

权限、持久化、钥匙串、剪贴板、弹窗、通知、联系人和日历。

permission统一查询权限状态，并请求位置或通知权限。storageUserDefaults 持久键值存储。databaseMiniApp 作用域内 SQLite 数据库和 JSON collection。keychainiOS 钥匙串安全存储。clipboard读取、写入和清空系统剪贴板。dialogs原生对话框、表单和日期选择。console原生弹窗、HUD 和控制台样式。notification本地通知权限、定时通知和角标管理。contacts联系人读取、编辑、选择器和 vCard 导入导出。calendar_events日历事件、提醒事项和 EventKit 权限。nfcNFC 标签读取和 NDEF 写入。
