#预期效果

#先选运行时

目标首选
写一个原生工具页、设置页、列表页或媒体控制页appui
写主屏、锁屏或 StandBy 小组件widget
迁移 Pythonista 风格命令式 UIui
写 2D 小游戏、Sprite、物理或逐帧绘制scene
调用照片、定位、通知、健康、蓝牙等系统能力对应原生模块 + AppUI 按钮
只运行脚本、处理文件或输出日志普通 Python + console

#失败路径

情况应该怎么处理
权限被拒绝显示当前状态和授权说明，不要循环弹出系统授权。
用户取消系统面板处理 None、False、空列表或错误字典，保持页面可继续操作。
设备或系统版本不支持显示不可用状态，并提供替代流程或禁用对应按钮。
返回数据为空把空状态写回界面，不把空结果当作异常崩溃。

#发布前检查

检查项合格标准
运行时页面型功能优先选 appui，小组件选 widget，游戏/绘制选 scene。
权限申请权限前先说明用途，并放到明确用户动作里。
数据普通设置用 storage，敏感数据用 keychain。
反馈成功、取消、失败和不可用状态都能在页面或 HUD 中看到。
文档跳转用户能从模块索引继续进入具体模块页和排错页。
