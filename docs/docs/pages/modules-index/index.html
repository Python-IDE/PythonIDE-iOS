#常见错误

错误写法后果修正
不选运行时直接用原生模块堆 UI难维护页面用 appui
import c_extensions模块不存在用真实包名
索引页当 API 文档签名错误点进具体模块页
权限未说明用途就申请被拒率高先展示说明再按钮申请
敏感数据用 storage不安全用 keychain
