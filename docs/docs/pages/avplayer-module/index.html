iOS 原生模块 / 媒体与视觉

avplayer

音视频加载、播放控制和原生播放器。

avplayer 是脚本级/底层原生 AVPlayer 能力模块，用于不写 AppUI 页面时加载远程或本地音视频、播放控制、跳转、调速、调音量和展示系统播放器。AppUI 页面里的视频控制优先用 AppUI 媒体控件 的 PlayerController。

#预期效果

示例会展示脚本级音视频加载、播放控制和播放失败处理。

#什么时候用

  • 播放一个远程或本地媒体 URL。
  • 非 AppUI 脚本需要多个命名播放器实例。
  • 需要 duration()、current_time()、seek(...)、set_rate(...) 等播放控制。
  • 视频需要系统播放器界面时调用 present_video()。

#不该用什么

情况应该改用
短 UI 音效sound 的 play_effect
多个短音效或本地音频文件对象sound 的 Player
AppUI 内嵌视频视图或页面内自定义控制AppUI 媒体控件 的 VideoPlayer / PlayerController

#核心模型

load(url) 返回布尔值，成功后才能调用 play()、pause()、seek() 或 present_video()。模块级函数管理默认播放器会话；切换页面、切换媒体或脚本结束时应调用 cleanup()。

Player(id="main") 管理命名播放器实例，支持与模块级函数相同的播放控制，并额外隔离进度、结束、错误、状态和 PiP 回调。它是底层脚本入口，不是 AppUI 新页面的首选写法。

on_progress(...) 和 on_end(...) 面向需要回调 id 的高级场景。普通脚本优先用 duration()、current_time() 读取状态。

#和 AppUI 的关系

AppUI 页面不要把 VideoPlayer 和 avplayer 混着控制。只展示视频时用 appui.VideoPlayer(url=...)；需要播放控制和事件时用 appui.PlayerController：

avplayer 保留给纯脚本、兼容旧代码和底层能力调用。

#失败路径

情况应该怎么处理
load(url) 返回 False不调用播放控制，显示加载失败。
duration() 为 0视为未知时长或媒体未准备好。
页面关闭或切换媒体调用 cleanup()。
视频需要系统播放器成功 load 后再调用 present_video()。

#常见错误

错误写法后果修正
play() 前没有 load()没有媒体可播放先检查 load(url)。
音频文件也 present_video()弹出不合适的播放器界面只对视频使用。
退出页面不清理旧媒体继续占资源调用 cleanup()。

#API 参考

类型API签名说明
functionloadload(url) -> Any加载媒体 URL 或本地路径，成功返回 True。
functionplayplay() -> Any开始或继续播放已加载媒体。
functionpausepause() -> Any暂停播放，保留当前位置。
functionstopstop() -> Any停止并回到开头。
functionseekseek(seconds) -> Any跳转到指定秒数。
functiondurationduration() -> Any获取总时长，单位秒。
functioncurrent_timecurrent_time() -> Any获取当前播放位置，单位秒。
functionis_playingis_playing() -> bool当前是否正在播放。
functionset_volumeset_volume(volume) -> None设置音量，范围 0.0 到 1.0。
functionset_rateset_rate(rate) -> None设置播放速度，1.0 为正常速度。
functionpresent_videopresent_video() -> Any展示系统 AVPlayerViewController。
functionon_progresson_progress(callback_id, interval_ms=500) -> Any注册进度回调。
functionstop_progressstop_progress() -> None停止进度回调。
functionon_endon_end(callback_id) -> Any注册播放结束回调。
functioncleanupcleanup() -> Any释放播放器资源。
classPlayerPlayer(id='default')命名原生播放器实例，方法包括 load、play、pause、stop、seek、duration、current_time、is_playing、set_volume、set_rate、on_progress、stop_progress、on_end、on_error、on_status、on_pip、cleanup。
