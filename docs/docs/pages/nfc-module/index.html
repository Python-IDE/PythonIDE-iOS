nfc

NFC 标签读取和 NDEF 写入。

nfc 用来读取 NFC 标签并写入 NDEF 记录。它需要真机、系统 NFC 支持和用户主动靠近标签；模拟器和不支持 NFC 的设备上应该显示不可用状态。

#预期效果

示例会展示 NFC NDEF 扫描、写入和设备不支持时的提示。

#什么时候用

  • 扫描一个 NFC 标签，读取其中的 NDEF 记录。
  • 写入短文本或 URL 到支持写入的 NFC 标签。
  • 做门禁、资产标签、设备配对等需要用户靠近标签的流程。

#核心模型

scan(...) 会启动系统 NFC 会话，返回 {"records": [...]} 或在取消、超时、失败时返回 None。write(records, ...) 写入 NDEF 记录，当前稳定记录类型是 text 和 uri。

NFC 会话必须由用户动作触发。不要在页面构建或脚本导入时自动扫描。

#失败路径

情况应该怎么处理
is_available() 为 False显示设备不支持，不显示扫描按钮或禁用按钮。
scan(...) 返回 None视为取消、超时或失败，保留当前页面。
write(...) 返回 False提示写入失败，允许用户重试。
标签不支持写入提示更换标签，不重试循环写入。
