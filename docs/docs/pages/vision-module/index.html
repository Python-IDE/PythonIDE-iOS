使用 Vision 从图片字节中识别文字。

使用 Vision 从图片字节中识别文字。

#预期效果

示例会展示图片 bytes OCR 的最小调用和空结果处理。

#适用场景

Vision OCR 入口，用于从图片字节中识别文字。

#标准示例

#AppUI OCR 配方

从相册选择图片做 OCR 时，用 photos.pick_image(raw_data=True) 获取 bytes，再传给 recognize_text_from_image_data(...)。选择器、识别和失败处理都放进按钮回调。

预期效果：点击按钮后打开相册选择器；取消、设备不可用和无文字都会显示明确状态，识别结果直接写回页面。

#API 参考

类型API签名说明
functionsetup_vision_frameworksetup_vision_framework() -> Optional[dict[str, Any]]初始化 Vision 框架并返回可用能力信息；初始化失败返回 None。
functionis_vision_availableis_vision_available() -> bool当前设备是否可用 Vision OCR。
functionrecognize_text_from_image_datarecognize_text_from_image_data(image_data: bytes) -> Optional[str]从图片 bytes 中识别文字，失败或无文字时返回 None。

#失败路径

情况应该怎么处理
权限被拒绝显示当前状态和授权说明，不要循环弹出系统授权。
用户取消系统面板处理 None、False、空列表或错误字典，保持页面可继续操作。
设备或系统版本不支持显示不可用状态，并提供替代流程或禁用对应按钮。
返回数据为空把空状态写回界面，不把空结果当作异常崩溃。

#使用规则

  • vision 只覆盖 OCR；人脸、条码、分类等用 vision_helper。
  • 传入图片 bytes，不是文件路径。
