这页按任务组织 widget API。需要确认参数名和返回值时，再进入 API 参考。

#最小结构

预期效果：生成一个最小可渲染小组件，只有标题和默认背景。

#选择规则

  • 信息卡、进度卡和列表优先用 row()、column()、grid()。
  • 表格、键盘、九宫格和海报式排版再用 table()、canvas() 和 .place()。
  • 可调标题、颜色、字号、文件和开关使用 widget.param。
  • 桌面点击必须通过 widget.state 或链接完成，不要写普通 Python 回调。
  • 会变化的文本、按钮和值默认加 .line_limit(1).min_scale(...)。

#失败路径

  • 找不到 API：先在本页按任务定位，再去 API 参考 查完整签名。
  • 示例能预览但桌面异常：看 排错，优先检查 w.render()、状态 action 和 WidgetKit 缓存。
  • 多尺寸表现不一致：回到 布局与尺寸，用 widget.family_value(...) 或 widget.context.is_family(...) 分支。
