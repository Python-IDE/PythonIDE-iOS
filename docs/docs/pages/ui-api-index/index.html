#失败路径

情况应该怎么处理
页面空白确认已创建根 ui.View，设置了非零 frame，并调用 present(...)。
点击无反应回调要赋函数对象，例如 button.action = tapped，不要写成 tapped()。
布局错位检查 frame、flex 和父视图尺寸；复杂响应式页面优先改用 appui。
输入或列表状态难维护把新页面迁到 appui.State、Form、List 和 NavigationStack。
